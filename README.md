# NARUA Artist Finder — MVMNTD INC

Dashboard interativo para prospecção de artistas do projeto NARUA.

## Como usar

1. Abra `index.html` no navegador — ou acesse via GitHub Pages
2. Cadastre artistas pelo formulário (nome, seguidores, engajamento, Spotify, etc.)
3. Ajuste os sliders de filtro na barra lateral
4. O score é calculado automaticamente e os artistas são ordenados do melhor para o pior perfil

## Score (0–100 pontos)

| Critério | Peso |
|---|---|
| Engajamento Instagram | 35pts |
| Views de Reels / seguidores | 25pts |
| Ouvintes mensais Spotify | 20pts |
| Frequência de postagem | 15pts |
| Penalidade — audiência comprada | -20pts |
| Bônus — já colaborou com selos | +5pts |

## Perfil ideal

- Score ≥ 65 pts → **Ideal**
- Score 40–64 pts → **Aceitável**
- Score < 40 pts → **Não recomendado**

## Hospedar no GitHub Pages

1. Crie um repositório no GitHub (ex: `narua-artist-finder`)
2. Suba esta pasta inteira
3. Vá em **Settings → Pages → Branch: main → / (root)**
4. Acesse em: `https://seuusuario.github.io/narua-artist-finder`
